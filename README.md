# HR Management System

A modern, responsive HR Management website built with React.js, Framer Motion, React Toastify, and Tailwind CSS.

## Features

### 1. Dashboard
- Quick action tiles (Approve leaves, Attendance, Payroll, Employees)
- Real-time attendance graph
- Notifications panel
- Upcoming events

### 2. Employee Management
- Employee directory with search
- Employee cards with contact info
- Add/view employee profiles

### 3. Attendance Management
- Daily attendance tracking
- Status indicators (Present, Late, Absent)
- Export functionality
- Statistics dashboard

### 4. Leave Management
- Leave request approval/rejection
- Leave statistics
- Real-time notifications

### 5. Payroll Management
- Salary structure display
- Payroll processing
- Downloadable payslips
- Auto-calculations

### 6. Performance Management
- KPI tracking
- Review scheduling
- Performance metrics

### 7. Recruitment Module
- Job openings management
- Applicant tracking
- Interview scheduling

## Tech Stack

- **React.js** - UI framework
- **Framer Motion** - Animations
- **React Toastify** - Notifications
- **Tailwind CSS** - Styling
- **Recharts** - Data visualization
- **Lucide React** - Icons
- **React Router** - Navigation
- **Vite** - Build tool
